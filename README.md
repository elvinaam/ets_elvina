# InnoBridge

**InnoBridge** adalah platform inovatif yang dirancang untuk menjembatani kesenjangan antara dunia akademik (mahasiswa) dan industri (perusahaan). Aplikasi ini memungkinkan perusahaan untuk mempublikasikan tantangan nyata yang mereka hadapi, sementara mahasiswa dapat menawarkan solusi kreatif sebagai bentuk kontribusi nyata dan pengembangan portofolio.

## 🚀 Deskripsi Aplikasi

InnoBridge hadir sebagai solusi digital untuk mengelola kolaborasi proyek. Fokus utama aplikasi ini adalah:
1. **Pusat Tantangan (Challenge Hub):** Tempat perusahaan mengunggah masalah teknis atau bisnis.
2. **Manajemen Proposal:** Memudahkan mahasiswa mengirimkan ide solusi dan portofolio.
3. **Tracking Progress:** Memantau sejauh mana kolaborasi berjalan antara kedua belah pihak.

---

## ✨ Fitur Utama
* **Dashboard Interaktif:** Tampilan ringkas untuk memantau challenge terbaru.
* **Submission System:** Fitur pengiriman file proposal dan link portofolio.
* **Role-Based Access:** Perbedaan tampilan untuk akun Mahasiswa dan akun Perusahaan.
* **Integration:** Terhubung dengan data pendukung untuk validasi proyek.
